To make the search case-insensitive, add "i" modifier to the pattern. You also should invert the result, since you do NOT need the word "password" to be present. 

const cond5 = !(/password/i.test(password));

And don't forget to add your new condition into array in return.

So, how my final solution should look like?

Here it is: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length > 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length > 9) cond2 = cond3 = true; // cond 4
    const cond5 = !(/password/i.test(password));
    
    return !([cond1, cond2, cond3, cond5].includes(false));
}
