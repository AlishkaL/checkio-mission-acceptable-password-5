Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Kontynuujmy z rozwiązaniem z podpowiedzi poprzedniej misji w serii. 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length> 9) cond2 = cond3 = true; // cond 4
    
    return!([cond1, cond2, cond3].includes(false));
}

W tej misji należy dodać do rozwiązania sprawdzenie, czy ciąg znaków zawiera słowo "password".

Od czego powinienem zacząć?

W ten sam sposób, w jaki poprzednio szukali Państwo cyfry i znaku niebędącego cyfrą, mogą Państwo szukać słowa "password" wewnątrz password. 

const cond5 = /password/.test(password);

Ok. Ale w ten sposób sprawdzam małe litery. Jak sprawdzić wielkie litery, mieszane?

Aby wyszukiwanie nie uwzględniało wielkości liter, proszę dodać do wzorca modyfikator "i". Proszę również odwrócić wynik, ponieważ słowo "password" NIE musi być obecne. 

const cond5 =!(/password/i.test(password));

I proszę nie zapomnieć dodać nowego warunku do tablicy w return.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Oto ono: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length> 9) cond2 = cond3 = true; // cond 4
    const cond5 =!(/password/i.test(password));
    
    return!([cond1, cond2, cond3, cond5].includes(false));
}
