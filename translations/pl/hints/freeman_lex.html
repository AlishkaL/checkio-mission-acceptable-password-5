Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Kontynuujmy z rozwiązaniem z podpowiedzi poprzedniej misji w serii. 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password)> 6
    cond2 = any(char.isdigit() for char in password)
    cond3 = any(char.isalpha() for char in password)
    if len(password)> 9: # cond4    
        cond2 = cond3 = True

    return all([cond1, cond2, cond3])

W tej misji należy dodać do tego rozwiązania sprawdzenie, czy ciąg składa się ze słowa "password" w każdym przypadku.

Od czego powinienem zacząć?

Aby sprawdzić, czy jeden ciąg znaków jest częścią drugiego, można użyć operatora in: 

'password' in password
# since you need word 'password' not to be present, above checking will be:
'password' not in password

Ok. Ale w ten sposób sprawdzam małe litery. Jak sprawdzić wielkie litery, mieszane?

Nie trzeba sprawdzać wszystkich przypadków. Proszę użyć metody string str.lower(). Przekształca ona ciąg znaków na małe litery i można sprawdzić obecność słowa tylko raz: 

cond5 = 'password' not in password.lower()
# you may transform string into uppercase with str.upper() as well. Then:
cond5 = 'PASSWORD' not in password.upper()

I proszę nie zapomnieć dodać nowego warunku do funkcji all() w return.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Oto ono: 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password)> 6
    cond2 = any(char.isdigit() for char in password)
    cond3 = any(char.isalpha() for char in password)
    if len(password)> 9: # cond4    
        cond2 = cond3 = True
    cond5 = 'password' not in password.lower()
        
    return all([cond1, cond2, cond3, cond5])

Rozwiązanie w odwrotnym stylu: sprawdzanie każdego warunku, jeśli nie jest True, natychmiastowy False powrót. Jeśli wszystkie True (to, czego potrzebujesz) - zwróć True na końcu: 

def is_acceptable_password(password: str) -> bool:

    if not len(password)> 6:
        return False
    if not len(password)> 9:
        if not any(x.isdigit() for x in password):
            return False
        if not any(x.isalpha() for x in password):
            return False
    if 'password' in password.lower():
        return False

    return True

Wszystkie warunki bezpośrednio w instrukcji return z wykorzystaniem modułu re do wyszukiwania cyfr i liter. Metoda find zwraca indeks początkowy lub -1, jeśli podciąg nie znajduje się w ciągu: 

import re


def is_acceptable_password(password: str) -> bool:

    return bool(
        (len(password)> 6 and re.search('\d', password) and re.search('[a-zA-Z]', password) 
        or len(password)> 9) 
        and (password.lower().find('password')) == -1)
