Давай продовжимо розв'язок з останньої підказки попередньої місії в серії. 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(char.isdigit() for char in password)
    cond3 = any(char.isalpha() for char in password)
    if len(password) > 9: # cond4    
        cond2 = cond3 = True

    return all([cond1, cond2, cond3])
У цій місії тобі слід додати до розв'язку перевірку, чи містить твій рядок слово 'password' у будь-якому регістрі.
З чого мені почати?
Щоб перевірити, чи є один рядок частиною іншого, тобі достатньо використати оператор 'in': 

'password' in password
# оскільки тобі потрібно, щоб слова 'password' не було, верхній рядок матиме вигляд:
'password' not in password
Добре. Але ж у такий спосіб я перевірю маленькі букви. Як перевірити великі чи змішане слово?
Немає потреби перевіряти це все. Використай рядковий метод str.lower(). Він перетворює всі літери в рядку на маленькі і ти можеш перевірити присутність слова лише раз: 

cond5 = 'password' not in password.lower()
# ти також можеш перетворити всі букви на великі з str.upper(). Тоді:
cond5 = 'PASSWORD' not in password.upper()
І не забудь додати твою нову умову в функцію all() в return.
То як мій остаточний розв'язок має виглядати?
Ось він: 

def is_acceptable_password(password: str) -> bool:

розв'язок у дещо протилежному стилі: перевірка кожної умови, якщо вона не True, негайне повернення False. Якщо всі умови True (те, що тобі потрібно) - повернення True в кінці: 

def is_acceptable_password(password:str)->bool:

    if not len(password) > 6:
        return False
    if not len(password) > 9:
        if not any(x.isdigit() for x in password):
            return False
        if not any(x.isalpha() for x in password):
            return False
    if 'password' in password.lower():
        return False

    return True
Всі умови відразу в виразі return з використанням модуля re для пошуку цифри і літери. Метод 'Find' повертає початковий індекс або -1 якщо рядок не є частиною іншого рядка: 

import re


def is_acceptable_password(password: str) -> bool:

    return bool(
        (len(password) > 6 and re.search('\d', password) and re.search('[a-z]', password) 
        or len(password) > 9) 
        and (password.lower().find('password')) == -1)
